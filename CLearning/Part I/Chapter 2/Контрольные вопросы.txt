4. Для чего предназанчена строка return 0 в программе "Hello World"?
	Для того, чтобы вернуть результат выполнения этой функции в точку вызова

5. Для чего предназначен компилятор?
	Для перевода исходного текста программы в объектный код

6. Для чего предназначена дерективы #include?
	Для подключения содержимого заголовочного файла

7. Что означает суффикс .h после имени файла в языке C++?
	Это означает, что файл является заголовочным, который содержит определения фунукций

8. Что делает редактор связей?
	Собирает воедино все файлы, которые были подключены дерективой #include

9. В чём заключается различие между исходным и объектными файлами?
	Исходный файл содержит в себе исходный текст программы
	Объектный файл содержит последовательность 0 и 1

10. Что такое интегрированная среда разработки и для чего она предназначена?
	IDE - это среда разработки, которая содержит в себе редактор кода, компилятор, отладчик

11. Если в книге всё понятно, то зачем нужна практическая работа?
